After filling all orders received for bound volumes of THE SENTINEL for 1891, we still have on hand a few in each style of binding. As there are only a limited number of these volumes those desiring there should order at once. The prices are the same as previously announced, namely, heavy manilla paper, $1.50; half roan (cloth, with leather back and corners), $2.50.
